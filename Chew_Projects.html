Scratch Project

Class:Computer Science and Software Engineering

The objective of this Scratch project was to make a game or story for others to play in Computer Science and Software Engineering. We were given a set of instructions and a time limit from 8/29/17 to 9/7/17. The teacher let us do what we pleased afterwards. My teammate and I decided to remix one of our favorite classic games, Galaga. First, we found a game from a random person that seemed good but could improve upon and made a copy of it. Afterwards my partner and I took turns coding and doing the project notebook. I learned a lot from this experience. I learned how to work well with others on coding, understand basic commands on scratch, and to be more open on expressing my ideas and opinions. One of the struggles we overcame was figuring out all of the code. The owner that coded this version of Galaga used a lot of different and complex blocks. In the end we found out how to manipulate the bugs in the game, the ship and its controls, and add a tutorial to the game. We were graded on this project using this rubric. Image Manipulation Project

Class: Computer Science and Software Engineering

The objective of this object was to create a photo editor that could fit the customer needs. We were given a set of instructions and about 2 weeks to complete this project. Our client was the family. The family wanted different shapes for borders and different colors for the borders. For this project, my partner and I decided that we wanted to do an ellipse border, a circle like border, and a regular border which could have the colors customized to your choosing. First we worked on the ellipse border and its color. We ended up choosing a teal colored border. We then moved on to the circle border. For that color we also chose teal. My partner tackled the regular frame by himself. He made the code so that you could enter in a color to the code and it would add a border of that color by expanding the image. Other than the last border, my partner and I always took turns coding and filling out the project notebook . We were graded using this rubric.
